🌐 5. PRÓXIMOS PASSOS NO DROPBOX DEVELOPER CONSOLE
─────────────────────────────────────────────────────────────────────────────

PASSO 1: Criar App
  1. Acesse https://www.dropbox.com/developers/apps
  2. Clique "Create app"
  3. Escolha:
     • API: Dropbox API
     • Access type: Full Dropbox
     • Type: Scoped access
  4. Clique "Create"
  5. ANOTE: App Key e App Secret

PASSO 2: Configurar Redirect URI
  1. Va em sua app → Settings
  2. Vá para "OAuth 2"
  3. Em "Redirect URIs", clique "Add"
  4. Digite: http://localhost:8000/oauth/callback
  5. Clique "Save"

PASSO 3: Executar Script para Refresh Token
  1. Abra terminal no backend:
     $ cd backend
     $ python app/get_dropbox_refresh_token.py
  
  2. Será solicitado:
     - DROPBOX_APP_KEY (cole o do passo 1)
     - DROPBOX_APP_SECRET (cole o do passo 1)
  
  3. Navegador abrirá para fazer login no Dropbox
  4. Você autoriza a app
  5. COPIA o DROPBOX_REFRESH_TOKEN gerado

PASSO 4: Adicionar em Replit Secrets
  1. Replit → Secrets (ícone de cadeado)
  2. Adicione 3 variáveis:
     • DROPBOX_APP_KEY = (valor do passo 1)
     • DROPBOX_APP_SECRET = (valor do passo 1)
     • DROPBOX_REFRESH_TOKEN = (valor do passo 3)
  3. Clique "Save"

PASSO 5: Reiniciar Backend
  1. Backend API → Restart
  2. Espere carregar (30-60 segundos)
  3. Verifique logs: "Dropbox OAuth inicializado com sucesso"

PASSO 6: Validar Conexão
  1. Acesse: http://localhost:8000/dropbox/test
  2. Resposta esperada:
     {
       "status": "success",
       "message": "Conexão com Dropbox validada",
       "entries_count": 42
     }
